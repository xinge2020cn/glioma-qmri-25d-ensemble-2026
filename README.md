# Glioma_qMRI_25D_Ensemble

The training and evaluation code for the 2.5D ResNet18 stacked ensemble model utilized in the study  
entitled *"Multimodal Quantitative MRI with 2.5D Deep Learning for Molecular Classification and Overall Survival Prediction in Adult-Type Diffuse Gliomas"* is available here.

This repository includes scripts for:
- Quantitative MRI preprocessing (MUSE-DWI, 3D-pCASL, syMRI)
- 2D / 2.5D / 3D deep-learning training and stacking ensembles
- Grad-CAM interpretability visualization
- Radiologist and radiomics feature benchmarking
- Kaplan–Meier and Cox regression survival analyses
